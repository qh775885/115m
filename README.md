# 115Master 野生版

本项目是基于 [cbingb666/115master](https://github.com/cbingb666/115master) 的 `115Master` 项目进行修改的版本

本项目修改了以下功能：

- 增加 上一集，下一集按钮。

- 增加 播放模式选择 播完暂停，循环播放，自动下一集

- 支持 画质 旋转 选择后记忆

- 移除 -调色 -收藏 -自动播放面板 -预览图开关 -云下载按钮恢复原生
  
- 修改播放列表完全到右侧
