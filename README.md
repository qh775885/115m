# 115m

本项目是基于 [cbingb666/115master](https://github.com/cbingb666/115master) 项目进行修改的版本

## 功能

- ✅ 上一集/下一集按钮
- ✅ 播放模式（播完暂停/循环播放/自动下一集）
- ✅ 画质/旋转记忆
- ✅ 收藏功能
- ✅ 播放列表右侧显示
- ❌ 移除：调色、自动播放面板、预览图开关、云下载按钮等等
